# 🗡️ 杀戮尖塔 2 - 比那名居天子 皮肤 Mod (Tenshi Hinanawi Skin)

> “天道行而无所依，地道行而无所不载！” —— 蔚蓝有顶天的大小姐降临尖塔！

本项目是为《杀戮尖塔 2》(Slay the Spire 2) 制作的皮肤替换 Mod。它将游戏中的原版角色 **铁甲战士 (Ironclad)** 极其华丽地全方位替换为《东方 Project》中的不良天人 —— **比那名居天子 (Tenshi Hinanawi)**！

## ✨ 核心特性 (Features)

本项目并非简单的贴图替换，而是从底层进行了极其硬核的逻辑劫持与动画重构，包含以下特性：

* 🎨 **全场景视觉替换**：
  * **选人界面**：专属大头贴、蔚蓝有顶天背景墙、专属角色介绍。
  * **战斗内**：重新映射的坐标系统，完美适配尖塔 2 的战斗视角。
  * **商店与篝火**：极其温馨的专属挂机动画，篝火旁甚至有 **永江衣玖 (Iku Nagae)** 作为伴侣陪同休息！
* 🎬 **全套定制动画状态机**：
  * 包含入场 (Intro)、待机 (Idle)、多种攻击 (Attack 1/2/3)、受击 (Hit 1/2/3)、施法/能力 (Cast 1/2/3)、胜利 (Victory) 以及极其安详的死亡 (Die) 动画。
* 📢 **沉浸式专属语音库 (Voice Lines)**：
  * **战斗入场**：极其傲娇的 "Ready" / "Go" 播报。
  * **战斗结算**：极其不屑的 "Tedium" (无聊/枯燥) 斩杀语音。
  * **动态战斗音效**：抽卡式的攻击、受击、施法语音随机播放，沉浸感拉满。
* ⚙️ **赛博镜像雷达同步 (底层黑科技)**：
  * 完美兼容《杀戮尖塔 2》第二幕的“帝王蟹”等夹击机制。天子机甲内置 0.05 秒级监听雷达，实现极其丝滑的前后转身同步。

## 📦 安装指南 (Installation)

1. 确保你已经安装了《杀戮尖塔 2》(Slay the Spire 2)。
2. 在 GitHub 的 [Releases 页面] (如果有的话，请在此处放上链接) 下载最新的 Mod 压缩包。
3. 将解压后得到的 `TenshiHinanawi` 文件夹（包含 `.dll`、`.json` 以及 `.pck` 文件）放入你的游戏 Mod 根目录：
   `你的游戏安装路径/Slay the Spire 2/mods/`
4. 启动游戏，在游戏内的 Mod 管理器中极其傲娇地勾选本 Mod 即可生效！

## 🛠️ 极客架构说明 (For Developers)

本项目的源码极其清爽，采用现代 C# 与 Godot.NET 编写，并利用 Harmony 库进行底层劫持。
为了保证极客级别的美学与执行效率，项目采用了极其严谨的模块化架构：
* `TenshiGlobals.cs`: 赛博数据中枢（包含所有的状态缓存、路径常量、随机数引擎与语音/动作对象池，以及极其优雅的按需懒加载协议）。
* `Patch_Combat.cs`: 战斗全生命周期劫持（入场语音、动画重定向、死亡拦截、结算全频段广播）。
* `Patch_Room.cs`: 特殊房间逻辑（商店完美同位体替换、篝火静谧结界）。
* `Patch_UI.cs`: 门面工程劫持（选人界面换脸与底层战术静音）。

如果你也想为《杀戮尖塔 2》开发底层逻辑 Mod，本仓库的架构极其欢迎你白嫖与参考！
